# Introduction

Sample application was created using Angular 2.

# Features

* Component
* Custom Directive
* Router module
* Http module
* Form module (template driven form approach)
* Services
* Rxjs Observable
* Animation

# How to start

```bash
npm install
# dev
ng s
```

# Running test

```bash
# In a single bash window
npm run test

# Debug - In two bash windows
npm run karma      # 1st window
npm run test.dev   # 2nd window
```
