# Features

 ```bash
* Component
* Router module
* Http module
* Form module (template driven form approach)
* Services
* Rxjs Observable
* Animation
```

# How to start

* First configure DataBase.
    ```bash
    server -> config.js
    ```
* Install all dependencies
    ```bash
    npm install
    ```
* run the project 
    ```bash
    npm start
    ```
    
# Eecommended Browser
    ```bash
    Google Chrome
    ```

# Running test

